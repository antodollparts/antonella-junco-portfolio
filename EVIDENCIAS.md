## Aprendizajes

### 1. ¿Qué fue lo más fácil y lo más retador?

Lo más fácil fue pensar en la idea de mi proyecto.

Lo más retador fue implementar correctamente las etiquetas y solucionar detalles de estilos CSS.

---

### 2. ¿Qué etiquetas semánticas usaste y por qué?

Utilicé etiquetas como:
- `<header>` para la parte del encabezado en la parte superior
- `<nav>` para la navegación
- `<main>` para el contenido principal
- `<section>` para dividir las secciones del contenido
- `<footer>` para el pie de página

Porque mejoran la accesibilidad, el SEO y la organización del código.

---

### 3. ¿Cómo organizaste tus commits?

Utilicé commits pequeños y descriptivos:
- feat: agrega estructura HTML y CSS
- docs: actualizar README y documentación del proyecto

---

### 4. ¿Qué mejorarías en la siguiente iteración?

- Agregar funcionalidades
- Optimizar imágenes
- Agregar modo claro

# Portafolio  — Página Web en HTML y CSS

Bienvenida/o a mi portafolio de desarrolladora de software.  
Este proyecto muestra mis habilidades técnicas, proyectos y conocimientos en desarrollo web y móvil.

## Instalación

1. Clona este repositorio:

   ```
   git clone https://github.com/antodollparts/antonella-junco-portfolio.git
   ```

2. Navega al directorio del proyecto:

   ```
   cd antonella-junco-portfolio
   ```

3. En Visual Studio Code descargar la extensión para visualizar:

   ```
   Live Server
   ```

4. Ejecuta la extensión:

   ```
   Visualiza el proyecto en tu navegador web
   ```

## Tecnologías utilizadas

- HTML5
- CSS3
